
# V3 Mail Settings Template Response

## Structure

`V3MailSettingsTemplateResponse`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `enabled` | `boolean` | Required | Indicates if the legacy email template mail setting is enabled. |
| `htmlContent` | `string` | Required | The HTML content of your legacy email template. |

## Example (as JSON)

```json
{
  "enabled": false,
  "html_content": "html_content4"
}
```
