
# CCBCC Email Object

## Structure

`CCBCCEmailObject`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `email` | `string` | Required | The intended recipient's email address. |
| `name` | `string \| undefined` | Optional | The intended recipient's name. |

## Example (as JSON)

```json
{
  "email": "jane_doe@example.com",
```
